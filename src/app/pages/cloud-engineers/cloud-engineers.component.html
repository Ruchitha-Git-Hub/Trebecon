Cloud Engineers

Dallas, Texas, USA | 04-06-2025

A cloud engineer is responsible for designing, implementing, and managing cloud-based infrastructure and services to support scalable, secure, and high-performance applications. This role involves working with cloud platforms, automation tools, and DevOps practices to ensure efficient cloud operations and business continuity.

Qualification: Any Graduation (B.Tech / B.E. in computer science or related field)

Experience: 3+ years

Shift Type: Afternoon shift

Location: Dallas, Texas, USA

Number of Openings: 5

Job Description :

Required Skills & Qualifications:

  • 3-5 years of experience in cloud engineering or cloud infrastructure management.
  • Hands-on experience with one or more cloud platforms: AWS, Microsoft Azure, or Google Cloud Platform (GCP).
  • Strong knowledge of cloud services such as EC2, S3, RDS, Lambda, Azure Functions, or GCP equivalents.
  • Experience with Infrastructure as Code (IaC) tools like Terraform, CloudFormation, or ARM templates.
  • Proficiency in scripting languages like Python, Bash, or PowerShell for automation tasks.
  • Experience with CI/CD pipelines and tools like Jenkins, GitLab CI, or Azure DevOps.
  • Understanding of networking, firewalls, DNS, load balancing, and VPNs in cloud environments.
  • Familiarity with containerization and orchestration tools such as Docker and Kubernetes.
  • Knowledge of monitoring, logging, and alerting tools such as CloudWatch, Prometheus, or Grafana.
  • Experience with security practices, compliance frameworks, and identity/access management in the cloud.

Workplace :

Work From Office

Roles & Responsibilities :

Key Responsibilities:

  • Design, deploy, and manage cloud infrastructure to support scalable and reliable applications.
  • Collaborate with development, operations, and security teams to implement cloud solutions.
  • Automate infrastructure provisioning using Infrastructure as Code (IaC) tools.
  • Ensure cost optimization, performance tuning, and availability of cloud resources.
  • Monitor cloud environments and proactively address issues related to performance, scalability, or security.
  • Implement CI/CD pipelines to streamline deployments and testing in cloud environments.
  • Participate in cloud migration efforts from on-premise or hybrid environments.
  • Stay up-to-date with the latest cloud services, tools, and best practices.
