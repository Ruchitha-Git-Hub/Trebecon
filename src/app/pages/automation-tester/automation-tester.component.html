Automation Tester

Miami, Florida, USA | 04-06-2025

An automation tester is responsible for designing, developing, and executing automated test scripts to ensure the quality and functionality of web and mobile applications. This role involves using automation tools and frameworks to streamline testing processes, identify defects early, and ensure robust software delivery.

Qualification: Any Graduation (B.Tech / B.E. in computer science or related field)

Experience: 3+ years

Shift Type: Afternoon shift

Location: Miami, Florida, USA

Number of Openings: 4

Job Description :

Required Skills & Qualifications:

  • 3-5 years of experience in automation testing for web and/or mobile applications.
  • Strong knowledge of automation tools such as Selenium, Appium, or Cypress.
  • Experience with test automation frameworks like TestNG, JUnit, or PyTest.
  • Proficiency in scripting languages such as Java, Python, or JavaScript.
  • Knowledge of continuous integration tools like Jenkins, GitLab CI, or CircleCI.
  • Experience with API testing tools like Postman, REST Assured, or SoapUI.
  • Familiarity with version control systems like Git and platforms such as GitHub/GitLab.
  • Understanding of Agile methodologies, SDLC, and QA best practices.
  • Ability to design test plans, test cases, and test scripts for both functional and non-functional testing.
  • Strong analytical and problem-solving skills.

Workplace :

Work From Office

Roles & Responsibilities :

Key Responsibilities:

  • Design, develop, and execute automated test scripts for functional, regression, and integration testing.
  • Collaborate with developers and QA teams to identify and resolve issues early in the development cycle.
  • Perform API testing and validate integrations with backend systems.
  • Ensure automation coverage for new features and maintain test scripts for existing functionality.
  • Continuously improve test frameworks and tools for efficiency and effectiveness.
  • Participate in sprint planning, stand-ups, and retrospectives as part of an Agile team.
  • Ensure quality standards are met throughout the software development lifecycle.
  • Stay updated with the latest trends and best practices in automation testing.
